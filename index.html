USB Route Finder

Web App

View project's page

This tool allows the Simón Bolivar University's Students to find the best route inside the campus to go from one point to other. It is very useful to the new students who are not familiarized with the location of the buildings where they have to take class.

This Web App was built with PostgreSQL (PostGIS + PGRouting), Geoserver and Leaflet. The lines and points that compose the Geometric Network was drawn in ArcMap 10.6. Behind the scene has code written in HTML, CSS, Javascrip and PHP.

Busca Tu Arepa

Web App

View project's page

"Busca Tu Arepa" is a Web App that pursuit to be a tool used by anyone who wants to find a Venezuelan restaurant. It was conceived principally for the Venezuelan Exodus could meet again with a piece of our culture.

This Web App was built with Leaflet and the data is dynamically consulted over a MySQL database. It has a sidebar in wich is populated the locals by category, the local information and the forms wich after been sends are written into the database. Also has a search box that use typeahead.js library.

Credits to Gabriel Lovera for his work building the database
